# POE2 WeGameBD增强

面向 WeGame《流放之路 2》BD 分享页的浏览器增强脚本。它会从 WeGame 公开 BD 分享网页获取技能信息，并在页面下方整理成更容易查看、复制和筛选的文字表格。

## 功能

## 安装入口

- 国际用户可从 GreasyFork 或 GitHub 安装。
- 国内用户如果打不开 GreasyFork / GitHub，可以从 Gitee 镜像安装：
  `https://gitee.com/aprilfool/Script/raw/main/Tampermonkey/POE2-WeGameBD-enhancer/POE2-WeGameBD-enhancer.user.js`
- 从 Gitee 安装的版本会优先使用 Gitee 地址检查更新，后续更新更适合国内网络环境。

### 技能表格化展示

- 在 BD 分享页技能区域下方插入「技能」面板。
- 将技能整理成清晰的卡片和表格，显示技能名称、类型、等级、品质、孔数、标签和需求。
- 将辅助石和嵌套主动技能统一放进「已插入技能」表格，方便横向对比。
- 用颜色区分主动、被动、精魂和装备赋予技能。
- 支持按技能类型筛选，按原顺序或类型排序。
- 支持一键复制完整技能文本，也可以手动选中文字复制。

### 多语言切换

- 支持简体中文、繁体中文、English 三种技能名称显示。
- 简体中文为默认语言。
- 繁体和英文名称来自 POE2DB 公开名称数据，仅用于本地显示转换。

## 适用页面

主要用于这类分享链接：

```text
https://www.wegame.com.cn/helper/poe2/#/share/xxxx
```

## 说明

- 脚本只读取 WeGame 公开 BD 分享网页上可获取的信息，不修改游戏数据。
- “被动技能”根据永久性、光环、捷等标签识别；只有公开页面数据里能明确看到精魂消耗或保留时，才显示为“精魂技能”。
- 等级达到 20 或以上时，如果页面数据没有写“最高等级”，脚本会自动补充“最高等级”显示。
- 主动技能品质为空或 `-` 时显示为 `品质0%`；表格里的被动辅助石没有品质时显示为 `-`。
- 如果公开页面数据没有提供某项技能需求，页面会隐藏该行或显示 `-`。
