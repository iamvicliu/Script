# POE2 NinjaBD增强

面向 poe.ninja《流放之路 2》BD 角色页的浏览器增强脚本。核心用途是更方便地抄作业：把公开角色页里的装备、药剂、珠宝和技能整理成文字表格，并支持英文、简体、繁体名称切换，方便对照国际服与国服的装备和技能名称。

## 功能

## 安装入口

- 国际用户可从 GreasyFork 或 GitHub 安装。
- 国内用户如果打不开 GreasyFork / GitHub，可以从 Gitee 镜像安装：
  `https://gitee.com/aprilfool/Script/raw/main/Tampermonkey/POE2-NinjaBD-enhancer/POE2-NinjaBD-enhancer.user.js`
- 从 Gitee 安装的版本会优先使用 Gitee 地址检查更新，后续更新更适合国内网络环境。

### BD 信息表格化展示

- 在 poe.ninja POE2 角色页底部插入「BD信息」面板。
- 将装备、药剂、珠宝整理成可复制的表格。
- 装备信息包含部位、名称、稀有度、属性/需求、插槽内容和词缀。
- 属性、插槽和词缀使用列表展示，长文本更容易逐条查看。
- 装备顺序按武器、副手、防具、首饰、腰带、药剂/咒符整理。
- 技能按卡片和表格展示，包含技能名称、类型、等级、品质、孔数、标签、需求和 DPS。
- 辅助石、血脉辅助和嵌套主动技能统一放进「已插入技能」表格，方便横向对比。
- 支持按技能类型筛选，按原顺序或类型排序。
- 支持一键复制完整文字，也可以手动选中文字复制。

### 多语言名称对照

- 支持 English、简体中文、繁体中文三种显示。
- English 为默认语言。
- 简体和繁体名称来自 POE2DB 公开名称数据，仅用于本地显示转换。
- 传奇装备、技能、辅助石等固定名称可用于跨服对照；稀有装备的随机名不会强行翻译，避免误导。
- 部分常见装备属性、需求、稀有度和词缀会做基础中文化，方便阅读。

### 镜像站支持

- 默认支持 `poe.ninja`、`poe.show`、`ninja.710421059.xyz`。
- 可在油猴脚本菜单里设置其他镜像站域名。
- 脚本会优先识别 poe.ninja 风格的 POE2 角色链接；镜像站设置只用于让面板在更多域名下启用。

## 适用页面

主要用于这类角色链接：

```text
https://poe.ninja/poe2/builds/<league>/character/<account>/<character>
https://poe.ninja/poe2/profile/<account>/<league>/character/<character>
```

镜像站只要保留类似路径，也可以通过油猴脚本菜单添加域名后启用。

## 说明

- 脚本只读取 poe.ninja 公开角色页面可获取的信息，不修改游戏数据。
- “被动技能”根据永久性、光环、捷、Buff 等标签识别；只有公开页面数据里能明确看到精魂消耗或保留时，才显示为“精魂技能”。
- Ninja 页面装备和词缀通常是英文；中文名称切换来自 POE2DB 名称映射，属性和词缀翻译只做辅助阅读。
- 如果公开页面数据没有提供某项信息，页面会隐藏该行或显示 `-`。
