# POE2 WeGame增强

面向 WeGame《流放之路 2》BD 分享页的辅助脚本，用来把页面里的技能配置转换成更容易阅读、复制和筛选的文字信息。

## 功能

- 在 WeGame POE2 BD 分享页技能区域下方插入「技能」面板。
- 自动读取 WeGame BD 接口技能数据。
- 每个技能卡片显示：
  - 技能名称
  - 技能类型：主动技能 / 被动技能 / 精魂技能（仅在接口能明确识别精魂消耗或保留时显示）
  - 技能等级
  - 品质
  - 孔数
  - 技能标签
  - 技能需求：等级、力量、敏捷、智慧等 WeGame 接口提供的需求字段
  - 已插入技能数量
  - 已插入技能表格
- 已插入技能用表格展示，包含序号、类型、名称、等级、品质、需求。
- 辅助石在表格里标记为“被动”；嵌套主动技能在表格里标记为“主动”，并显示自身等级、品质、孔数和需求。
- 主动、被动、精魂、装备赋予技能使用不同颜色标记；表格里的主动 / 被动类型也有颜色区分。
- 顶部提供技能汇总：
  - 总技能数
  - 主动技能数
  - 被动技能数
  - 可确认的精魂技能数
- 支持筛选：
  - 全部
  - 主动
  - 被动
- 支持排序：
  - 原序
  - 类型
- 支持技能名称语言切换：
  - 简体中文（默认）
  - 繁体中文
  - English
- 繁体和英文名称从 POE2DB 名称数据加载。
- 语言、筛选、排序按钮会显示当前选中状态。
- 支持一键复制完整技能文本。
- 支持手动选择面板文字复制。

## 适用页面

```text
https://www.wegame.com.cn/helper/poe2/*
```

主要用于这类分享链接：

```text
https://www.wegame.com.cn/helper/poe2/#/share/xxxx
```

## 说明

- 脚本只读取公开 BD 分享页已有的 WeGame 接口数据，不修改游戏数据。
- “被动技能”根据永久性、光环、捷等标签识别；只有接口字段里能明确看到精魂消耗或保留时，才显示为“精魂技能”。
- 若 WeGame 接口未提供可靠的装备来源字段，脚本不会在筛选里单独展示装备赋予技能，避免误导。
- 等级达到 20 或以上时，如果接口没有写“最高等级”，脚本会自动补充“最高等级”显示。
- 主动技能品质为空或 `-` 时显示为 `品质0%`；表格里的被动辅助石没有品质时显示为 `-`。
- 技能需求来自 WeGame 技能接口字段；如果接口没有返回需求，页面会隐藏该行或显示 `-`。
- 面板会嵌入在 WeGame 原技能区域下方，并尽量保持与原页面布局一致。
- 首次切换繁体或英文时需要请求 POE2DB 名称数据，之后会直接复用本页已加载的数据。
- 如果 POE2DB 某个语言数据临时加载失败，脚本会保留已成功加载的语言缓存；繁体名称还会使用简繁转换兜底。
