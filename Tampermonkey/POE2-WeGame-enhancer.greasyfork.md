# POE2 WeGame增强

面向 WeGame《流放之路 2》BD 分享页的辅助脚本，用来把页面里的技能配置转换成更容易阅读、复制和筛选的文字信息。

## 功能

- 在 WeGame POE2 BD 分享页技能区域下方插入「技能文字信息」面板。
- 自动读取 WeGame BD 接口技能数据。
- 每个技能卡片显示：
  - 技能名称
  - 技能类型：主动技能 / 精魂技能
  - 技能等级
  - 品质
  - 孔数
  - 技能标签
  - 辅助石数量
  - 辅助石表格
- 辅助石用表格展示，包含序号、名称、等级、品质，方便对齐查看每个技能插了几个辅助。
- 顶部提供技能汇总：
  - 总技能数
  - 主动技能数
  - 精魂技能数
  - 总辅助石数
  - 当前显示数量
- 支持筛选：
  - 全部
  - 主动
  - 精魂
  - 有辅助
  - 无辅助
- 支持排序：
  - 原序
  - 类型
- 支持一键复制完整技能文本。
- 支持手动选择面板文字复制。

## 适用页面

```text
https://www.wegame.com.cn/helper/poe2/*
```

主要用于这类分享链接：

```text
https://www.wegame.com.cn/helper/poe2/#/share/xxxx
```

## 说明

- 脚本只读取公开 BD 分享页已有的 WeGame 接口数据，不修改游戏数据。
- “精魂技能”根据技能标签中的永久性、光环、捷等信息识别。
- 若 WeGame 接口未提供可靠的装备来源字段，脚本不会在筛选里单独展示装备赋予技能，避免误导。
- 等级达到 20 或以上时，如果接口没有写“最高等级”，脚本会自动补充“最高等级”显示。
- 品质为空或 `-` 时显示为 `Q+ 0%`。
